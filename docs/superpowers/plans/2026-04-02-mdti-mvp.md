# MDTI MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** CLAUDE.md를 브라우저에서 분석하여 페르소나를 분류하고, 발칙한 로스팅 결과를 LinkedIn으로 공유할 수 있는 소셜 바이럴 웹 서비스를 만든다.

**Architecture:** 단일 Next.js App Router 앱. 분석 엔진은 클라이언트 사이드 JS로 실행되어 원본 MD가 서버를 거치지 않음. 결과만 Supabase에 저장하고 공유 URL + OG 이미지를 생성. Vercel 배포.

**Tech Stack:** Next.js 15 (App Router), TypeScript, Tailwind CSS 4, Supabase (PostgreSQL), @vercel/og (Satori), nanoid, Vercel
